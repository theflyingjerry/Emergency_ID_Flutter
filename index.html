Introduction

Welcome to Emergency ID!

Emergency ID is a mobile application designed by Max Phoenix (and team). The application aims to utilize the new norm of everyone having cellphones in order to help in medical emergencies. With the Emergency ID app, the user will have a simple yet extremely useful list of information regarding identification, medical issues, and whom ro contact in case of emergency. Simply opening the app could be the difference for someone as the app will allow helpers to quickly contact the neccesary parties and quickly inform medical responders of any neccesary knowledge, leading to better treatment.

Usage

Short App Usage Description

First Time Use

When the user first opens the Emergency Id application an entry page will de displayed. The user will be prompted to fill in a series of text entry feild corresponding to the neccesary section, for example allergies. The entry page is diveded into two sections, the personal information and the emergency contact information. The entry feild values are intially set to "N/A" and if the user does not change or leaves a feild blank the value will be saved as "N/A" otherwise the new information will be recorded. For example when prompted to input information regarding allergies the user could leave the feild blank for "N/A" or input their allergies so medical responders would know in case of emergency.

The user can the enter their emergency contacts' information. The contact information includes the contact name and their phone number. The app initially gives the user two emergency contacts, however the user can later edit this and add up to two more contacts or remove contacts. If the user leaves a contact information blank it will not be added, and if the user leaves both contact information feilds blank the app will just automatically add 911 in place of an emergency contact.

Once the user has entered all their desired information the main app will open and display the information.

Normal Usage

On every subsequent opening of the app, the entry page will not be displayed and instead the app will load the main display page and show all the users vital emergency information. This app display can be seen in the following section. The main display shows the user name at the top for easy identification followed by the user's personal information. The personal information shows the user's medical issues and allergies at the top fo the section and then one can scroll to see more information like the user's date of birth and phone number and email (useful if the phone is found lost). The personal section may show more based on the phone's height. Below the personal information the contact information is displayed, showing the contact's name and then phone number.

Calling

To call a contact simplfy press the listed phone number on the display.

Texting

To send a prewritten emergency text alerting the contact long press the listed number.

Email

To email the user if phone is found lost simplfy press the listed email in thw personal section of on the display. The app will redirect to the native email application.

Settings and Settings Page

The user can access the settings page, which is the same as the initial entry page, by swiping to the right on the display. The settings page will the be loaded with the current inforation loaded into the entry feilds. If the user wants to edit or remove the information they can easily click on the entry feild and edit the text. The user can also edit the contact information or amount of contacts. There is a floating action button at the bottom right hand corner of the display. Pressing the button will add a set of additional contact entry feilds. The user can also remove unwanted additional contacts by swiping right on either the contact name or contact phone number feild. Once the user is satisfied with the chages to the infomration they can press a back button at the top left hand corner of the display and return the the main display with the updated information.

                      } else if (phoneNumber == '911') {
                        phoneNumberDisplay = '911';
                      } else {
                        phoneNumberDisplay = phoneNumber;
                      }
                      return phoneNumberDisplay;
